数据导入:

通过命令行操作数据库:

python manage.py shell:


>>> from blog.models import StudentPost
>>> stu2 = StudentPost()
>>> stu2.name = "谢果"
>>> stu2.sex = "women"
>>> stu2.save()
>>> StudentPost.objects.all()
[<StudentPost: StudentPost object>, <StudentPost: StudentPost object>]



从上图可知:

定义表结构一定要在models里定义，增加和删除数据直接数据库的客户端工具修改即可。
